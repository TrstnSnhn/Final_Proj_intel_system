# PlantGuard AI

PlantGuard AI is an end-to-end plant disease classification project for the 6INTELSY Final Project (AY 2025–2026, Holy Angel University).

## Status
Project scaffold initialized. Core training/evaluation pipeline, CNN/NLP/RL components, and documentation templates are included.

## Quick Start
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
bash run.sh
```

If you hit a missing-package error (for example `ModuleNotFoundError: No module named 'PIL'`), rerun:

```bash
python -m pip install -r requirements.txt
```
